== Installation ==

This section describes how to install the plugin and get it working.

1. Download and extract the tawkto.zip
2. Upload `tawkto` directory to the `/wp-content/plugins/` directory
3. Activate the `Tawk.to Live Chat` plugin through the 'Plugins' menu in WordPress
4. Go to `Settings`, then to `Tawk.to` sub menu, and paste the embed code in the text area.

Note: You will need a free tawk.to account to generate the embed code, <a href="https://tawk.to/?utm_source=wpdirectory&utm_medium=link&utm_campaign=signup">Create one for free here!</a>.
